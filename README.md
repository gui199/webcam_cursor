# follow_color
Move mouse cursor by following specific color from your webcam

![Screenshot 1][screenshot]

# Install

## Ubuntu - install dependencies
    $ cd install/ubuntu
    $ ./install_dependencies.sh


# Run
    $ cd color_picker
    $ ./run.sh
